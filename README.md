### README.md ###

CMS-headless pour gérer le backend: https:https://c3f93ot0.directus.app
pour l'accès admin, se référer à la copie de l'ECF

le site est codé en HTML, CSS3 et javascript vanilla.

NodeJS a été utilisé pour directus mais n'est pas nécéssaire pour le front.
Il n'y a donc pas de npm install à faire.

Il suffit de récupérer les sources en local ou de les transférer via un logiciel FTP (ex filezilla) sur un serveur pour utiliser le site en lancant le fichier index.html.
